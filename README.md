## android + flask 后端

使用方式：

1. clone至本地

   ```
   git clone https://github.com/KongWiki/SimpleServer.git
   ```

2. 创建虚拟环境

   ```
    virtualenv env
    source env/bin/activate
   ```

3. 安装所需依赖

   ```
   pip install -r requirements.txt
   ```

4. 运行

   ```
   python manage.py runserver
   ```
